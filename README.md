# ML-Assignment2

## Overview

This project explores interpretability in convolutional neural networks using Class Attribution Maps (CAM) and feature‑map visualizations. A pretrained ImageNet model (ResNET-18) is used to analyze how different layers respond to images and how specific regions contribute to class logits.

### Good to know

"report.ipynb" contain all the code and analysis.

The file "pictures" contain all the examples used in the notebook. 

Note that the class index file is not part of the repo, as it was provided on the course website.
